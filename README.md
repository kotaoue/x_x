Got more to say than the character limit allows? Paste your text, pick a limit, and x_x splits it into post-sized chunks at natural break points — no more copy-paste headaches.

## Example

**Input** (limit: 140 chars):

```
Today I tried making homemade sushi for the first time! It was a total disaster at first. The rice was too sticky and the rolls kept falling apart. But by the end I had something edible!
```

**Output** — split into 2 posts:

```
[1/2] Today I tried making homemade sushi for the first time!
```
```
[2/2] It was a total disaster at first. The rice was too sticky and the rolls kept falling apart. But by the end I had something edible!
```
